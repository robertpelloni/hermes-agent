# Hermes Agent Vision

## Long-Term Product Goal
Hermes Agent is designed to be the ultimate self-improving AI agent. It aims to transcend traditional static conversational bots by incorporating a built-in learning loop. The agent should be able to create skills from experience, improve them during use, nudge itself to persist knowledge, search its own past conversations, and build a deepening model of the user across sessions.

## Design Direction
- **Omnipresent & Decoupled:** Live where the user does (Terminal, Telegram, Discord, Slack, etc.) and run anywhere (local, Docker, VPS, modal, serverless).
- **Model-Agnostic:** Support an arbitrary number of LLM providers without code changes or lock-in.
- **Autonomous Learning:** Automatically extract skills from interactions and iteratively refine them.
- **Research-Ready:** Support batch processing, reinforcement learning environments, and trajectory compression for AI research.
