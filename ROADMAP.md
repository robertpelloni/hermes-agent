# Hermes Agent Roadmap

## Future Focus Areas
1. **Windows Support Stabilization**
   - Move native Windows support from early beta to stable.
2. **Expand Multimodality**
   - Better out-of-the-box support for vision, audio (STT/TTS), and complex tool integration.
3. **Advanced Skill Orchestration**
   - Further refine the autonomous skill creation (Curator) and inter-skill communication.
4. **Agent Collaboration**
   - Enhance Kanban features for richer multi-agent workflows and task delegation.
